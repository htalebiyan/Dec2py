Judgment Call
=======================================================
These modules and classes model a multi-agent decision-making environment where each agent has the authority to restore a single layer, has complete information about her respective layer, and minimal or no information about other layers. It is assumed that agents communicate poorly and not on time. The decentralized methods is called Judgment Call (JC)  :cite:`Talebiyan2019c,Talebiyan2019` with and without auction-based resource allocations :cite:`Talebiyan2020a`.
